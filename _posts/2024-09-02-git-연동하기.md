### Git 연동하기
* `git clone` 받은 후 
* `clone` 받은 디렉토리 이동 후 아래 명령어 실행
```
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git remote add origin [GIT URL]
$ git push -u origin main
```
* 로컬환경에 `git`이 설치되어있는 경우에 위 명령어 실행 가능
